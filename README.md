# task2
Deployment of task2 on Hasura 

Steps to be followed:

1. Run git clone in console.
2. Run cd task1.
3. Run npm install
4. Run npm start
5. Open http://localhost:8080 in your favourite browser.

Endpoints:

1. '/'

2. '/authors'

3. '/image'

4. '/setcookie'

5. '/getcookies'

6 '/robots.txt'

7 '/input'
